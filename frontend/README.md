# News Headline Generator - Frontend

A clean, classic, React-based web interface for the Hugging Face `news-headline-generator` model.
This frontend supports generating news headlines directly from articles, predicting one or multiple headlines per article, and includes a fallback/improvement feature powered by the Gemini API for incomplete headlines.

## Features Let's Create
1. **1 Article → 1 Headline:** Instantly generate a short headline from a single article.
2. **1 Article → Multiple Headlines:** Generate up to 5 varied headline choices for a single article.
3. **Multiple Articles → 1 Headline Each:** Input multiple articles at once and process them sequentially for bulk generation.
4. **Gemini API Headline Improvement:** A key to invoke Google Gemini to fix and refine headlines generated by the HF model (which might end prematurely or miss critical grammar due to its training data).

## How to use
You require specific API keys to use this application:

### 1. Hugging Face Access Token
Required for generating initial headlines.
1. Sign up/Log in to [Hugging Face](https://huggingface.co/)
2. Go to Settings -> Access Tokens (or [this link](https://huggingface.co/settings/tokens))
3. Create a new token with "Read" access.

### 2. Google Gemini API Key (Optional but recommended)
Required for the "Fix & Improve" feature.
1. Go to [Google AI Studio](https://aistudio.google.com/)
2. Get API key.
3. Create a `.env` file in the `frontend` folder and add your key:
   ```env
   VITE_GEMINI_API_KEY=your_actual_api_key_here
   ```

## Installation & Running

1. Open a terminal and navigate to the `frontend` folder:
   ```bash
   cd frontend
   ```
2. Install the necessary dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open the web page shown in the terminal (usually `http://localhost:5173/`).
5. Click **API Settings** at the top right, enter your keys, and start generating!
